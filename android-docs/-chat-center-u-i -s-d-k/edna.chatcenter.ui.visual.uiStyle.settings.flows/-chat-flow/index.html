Parameters

Стиль для тулбара. Содержит стили для кнопки "назад", "копировать", "меню" и т.д.

Если вы встраиваете чат на весь экран, данный параметр должен быть включен для Android 15 и выше, т.к. он добавляет отступ согласно "edge-to-edge" подходу для нижнего навигационного бара (по умолчанию = true)
