TextImageComponent

data class TextImageComponent(context: Context, colors: ChatColors, texts: ChatTexts, images: ChatImages)

Компонент для текста вместе с изображением (например, кнопки камеры, галереи и выбора файлов в нижнем меню). Предоставляет доступ к более низкому уровню персонализации, а также общие настройки для группы параметров.
