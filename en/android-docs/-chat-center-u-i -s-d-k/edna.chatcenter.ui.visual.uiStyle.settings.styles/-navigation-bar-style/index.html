Parameters

backgroundColor

фоновый цвет (ресурс)

statusBarColor

цвет статус бара (ресурс)

isStatusBarLightIcons

определяет цвет иконок в статус баре. По умолчанию null - цвет берется в зависимости от системной темы

visibleShadow

определяет видимость тени под тулбаром

centerToolbarText

определяет центрирование текста в тулбаре

closeButtonEnabled

отображение кнопки "Назад"

chatSubtitleEnabled

устанавливает видимость подзаголовка в тулбаре

backButton

стиль для кнопки "назад"

invertedBackButton

стиль для инвертированной кнопки "назад" (если показываются кнопки "копировать", "цитата")

searchButton

стиль для кнопки "меню"

copyButton

стиль для кнопки "копировать"

replyButton

стиль для кнопки "ответить с цитированием"

toolbarHeaderText

стиль для заголовка в тулбаре (в случае обычной TextView)

toolbarSubtitleText

стиль для подзаголовка в тулбаре

toolbarSpannableHeaderText

стиль для заголовка в тулбаре (в случае SpannableText)
